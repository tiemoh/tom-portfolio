How It Works

Each marker exists on both boards under the same player id, so you can mirror positions or deliberately separate them by switching linked movement off.

Squad data is pulled client-side from TheSportsDB. That is enough for a portfolio demo; if you later add a commercial scouting API, route it through your own backend function.
